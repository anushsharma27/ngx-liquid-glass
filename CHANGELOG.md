# Changelog

All notable changes to `ngx-liquid-glass` are documented here. The project follows [Semantic Versioning](https://semver.org/) and the format from [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

### Fixed

- Consumer compatibility CI now installs the packed npm tarball, matching real-world installation and dependency resolution.

## [0.0.6] - 2026-07-19

### Added

- Angular 20, 21, and 22 compatibility, documented and verified through consumer builds.
- Continuous integration checks for production builds, unit tests, package contents, dependency auditing, and supported Angular versions.
- Contributor, security, support, and community health documentation.

### Changed

- Development workspace upgraded from Angular 19 to Angular 20.
- Removed unused Angular workspace dependencies.
- Expanded package documentation with selection guidance, complete API inputs, capability-based browser behavior, accessibility notes, package footprint, limitations, and support expectations.

## [0.0.5] - 2026-07-11

### Changed

- Improved package metadata and search discovery signals.
- Updated demo sitemap metadata.

## [0.0.4] - 2026-07-10

### Changed

- Expanded npm keywords and Angular compatibility metadata.
- Improved sitemap entries for the live demo.

## [0.0.3] - 2026-07-10

### Changed

- Moved published package and demo links to the canonical GitHub repository.

## [0.0.2] - 2026-07-10

### Fixed

- Corrected published repository, demo, and screenshot links.

## [0.0.1] - 2026-07-09

### Added

- Initial release of the standalone `NgxLiquidGlassDirective`.
- Configurable intensity, radius, tint, refraction, backdrop, and disabled inputs.
- SSR-safe browser integration and graceful browser fallbacks.
- `ng add` installation schematic and interactive demo.
