# ngx-liquid-glass workspace

Angular 19 workspace for the `ngx-liquid-glass` publishable library and its demo app.

**[Open the live demo](https://anushsharma27.github.io/ngx-liquid-glass/)**

## Projects

- `projects/ngx-liquid-glass`: standalone directive library.
- `projects/demo`: live demo app with controls for intensity, radius, tint, and refraction.

## Commands

```bash
npm install
ng build ngx-liquid-glass
ng test ngx-liquid-glass --watch=false --browsers=ChromeHeadless
ng build demo
ng serve demo
```

The library package README is at `projects/ngx-liquid-glass/README.md`.
