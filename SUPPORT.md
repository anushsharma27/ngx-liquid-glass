# Support

## Getting help

Before requesting help:

1. Read the [package documentation](projects/ngx-liquid-glass/README.md).
2. Check the documented browser behavior and known limitations.
3. Search [existing issues](https://github.com/anushsharma27/ngx-liquid-glass/issues).
4. Reproduce the problem with the latest package version and a supported Angular version.

Use the repository's issue forms for reproducible bugs and focused feature requests. General Angular, CSS, or browser questions are better suited to their respective community support channels.

This is a community-maintained open-source project. Response times are not guaranteed. Please do not use public issues for security vulnerabilities; follow [SECURITY.md](SECURITY.md).
